# icon 图标

## 代码演示

### 基础用法

通过 `name` 设置图标名。

```vue
<s-icon name="close" />
```

### 大小

通过 `size` 设置图标尺寸。

```vue
<s-icon size="20px" />
```

### 颜色

通过 `color` 设置图标颜色。

```vue
<s-icon color="#333" />
```

### API

| 参数  | 说明                  | 类型   | 默认值  |
| ----- | --------------------- | ------ | ------- |
| name  | 图标名                | String | -       |
| size  | 图标大小,如 20px, 2em | String | -       |
| color | 图标颜色              | String | inherit |
